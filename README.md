# NBA 2K Rating Predictor

Проект для анализа статистики игроков NBA и предсказания их рейтинга в NBA 2K.

## Что делает проект

Проект:

- загружает датасет с игроками NBA;
- удаляет лишние колонки;
- строит несколько графиков для анализа данных;
- обучает модели линейной регрессии;
- сравнивает качество моделей по RMSE;
- показывает, какие признаки сильнее всего связаны с рейтингом.

## Данные

Датасет лежит здесь:

```text
data/nba_rankings_2014-2020.csv
```

Целевая колонка:

```text
rankings
```

То есть модель пытается предсказать рейтинг игрока в NBA 2K.

## Что удаляется из признаков

Из модели убраны:

- `Unnamed: 0` — технический индекс;
- `PLAYER` — имя игрока;
- `rankings` — target;
- `SEASON` и `TEAM` — категориальные текстовые признаки;
- `FP` — fantasy points, потому что это само по себе является некоторой моделью.

## Графики

В проекте строятся:

- распределение рейтингов;
- корреляционная матрица;
- график `PTS vs rankings`;
- boxplot рейтингов;
- важность признаков Ridge;
- распределение ошибок модели;
- сравнение моделей по RMSE.

## Модели

Используются:

- baseline-модель, которая предсказывает средний рейтинг;
- обычная линейная регрессия;
- линейная регрессия после масштабирования признаков;
- линейная регрессия с подобранным `alpha`;
- Ridge после удаления самых больших ошибок на обучении.

## Как запустить

Создать виртуальное окружение на Windows:

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

Создать виртуальное окружение на Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить проект:

```bash
python main.py
```

## Структура проекта

```text
main.py              # запуск проекта
src/config.py        # настройки
src/data.py          # загрузка и подготовка данных
src/modeling.py      # обучение моделей
src/plots.py         # графики
src/reporting.py     # итоговые отчеты
data/                # датасет
```

## Основная идея

Рейтинг NBA 2K сильнее всего связан с базовой статистикой игрока:

- очки;
- попадания с игры;
- подборы;
- передачи;
- минуты;
- победы;
- блок-шоты;
- перехваты.
